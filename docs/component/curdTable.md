## API

| 属性         | 说明                                         | Type   | 默认值 |
|--------------|--------------------------------------------|--------|--------|
| name         | 该页面的名称，用于表格上部按钮添加，删除的名称 | string | -      |
| tableConfig  | 表格配置，详见 [table](table.md) 组件         | Object | -      |
| searchConfig | 搜索配置，详见 [search](search.md) 组件       | Object | -      |
| formConfig   | 表格配置，详见 [submit](submit.md) 组件       | Object | -      |
